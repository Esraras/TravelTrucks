# TravelTrucks

## Kısa Açıklama
TravelTrucks, kullanıcıların farklı karavanları filtreleyip inceleyebileceği, detaylarını görebileceği ve favorilerine ekleyebileceği modern bir React uygulamasıdır.

## Temel Özellikler
- Karavan kataloğu ve detay sayfaları
- Konum, ekipman ve araç tipine göre filtreleme
- Favorilere ekleme
- Responsive ve kullanıcı dostu arayüz

## Kurulum ve Kullanım
1. Depoyu klonlayın:
   ```bash
   git clone <repo-url>
   cd travelTrucks
   ```
2. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```
3. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```
4. Uygulamayı tarayıcıda görüntüleyin: `http://localhost:5173`
